Dentro da pasta **src**, crie um arquivo chamado **youtubeUrls.js**, contendo as urls dos videos que deseja baixar.

Exemplo:

```bash
const youtubeUrls = [
  "https://www.youtube.com/watch?v=E0ozmU9cJDg&ab_channel=Metallica-Topic",
];

module.exports = youtubeUrls;
```
